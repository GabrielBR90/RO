# Meu App Android

Este é um aplicativo Android simples criado em Kotlin. Ele exibe uma mensagem "Olá, Usuário" ao abrir.

## Como Instalar
1. Baixe o APK na seção de releases.
2. Instale no seu dispositivo Android.

## Tecnologias Usadas
- Kotlin
- Android SDK
- Gradle
